# CausalAnalysisforSemiCompRisks
An R-package for analyzing semi-competing risks data using causal mediation inference. This package is aimed to illustrate the methodology presented in "Semiparametric causal mediation modeling of hepatitis on mortality through liver cancer incidence".

## Installation
```r
devtools::install_github("JuShengHong/CausalAnalysisforSemiCompRisks")
```
